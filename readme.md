# Kanji Flashcards Game
A simple web-based game to help users learn JLPT N4 Kanji. Guess the reading of the Kanji, and get instant feedback on your answer, including the correct reading and meaning.

## Features:
*   Interactive flashcards for JLPT N4 Kanji.
*   Guess the reading (Onyomi/Kunyomi).
*   Instant feedback with correct reading and meaning.
*   Score tracking.
*   10-question quiz format.
*   "Play Again" functionality.

## How to Play:
1.  Open `index.html` in your web browser (preferably served via a local HTTP server to avoid CORS issues).
2.  Click the "Play" button to start the quiz.
3.  Enter your guess for the Kanji's reading in the input field.
4.  Click "Submit" or press Enter.
5.  Click "Next" to proceed to the next Kanji.
6.  After 10 questions, your final score will be displayed.

# 漢字フラッシュカードゲーム
JLPT N4の漢字学習を支援するシンプルなウェブベースのゲームです。漢字の読み方を推測し、正解の読み方と意味を含む即座のフィードバックを受け取ることができます。

## 特徴:
*   JLPT N4漢字のインタラクティブなフラッシュカード。
*   読み方（音読み・訓読み）を推測。
*   正解の読み方と意味を含む即座のフィードバック。
*   スコア追跡機能。
*   10問のクイズ形式。
*   「もう一度プレイ」機能。

## 遊び方:
1.  ウェブブラウザで `index.html` を開きます（CORSの問題を避けるため、ローカルHTTPサーバー経由で開くことをお勧めします）。
2.  「Play」ボタンをクリックしてクイズを開始します。
3.  入力フィールドに漢字の読み方の推測を入力します。
4.  「Submit」をクリックするか、Enterキーを押します。
5.  「Next」をクリックして次の漢字に進みます。
6.  10問終了後、最終スコアが表示されます。
